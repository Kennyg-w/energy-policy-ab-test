# Energy Policy A/B Test (Statistical Significance Project)

## 📊 Project Overview
Designed a statistical experiment to determine if a new "Peak-Hour Reward" policy significantly reduces residential energy consumption.

## 🛠️ Technical Achievements
- **Experimental Design:** Developed a Control/Treatment framework for 2,000 simulated energy users.
- **Inferential Statistics:** Implemented a two-sample Independent T-Test using `scipy.stats`.
- **Significance Testing:** Achieved a **P-Value of 0.0021**, providing high-confidence proof of policy effectiveness.
- **Business Logic:** Calculated a **4.33% reduction** in mean energy usage, enabling a data-driven ROI analysis for policy rollout.

## 🐍 Tech Stack
- Scipy (Stats), Pandas, Seaborn, Matplotlib.
